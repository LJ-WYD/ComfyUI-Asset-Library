# ComfyUI Asset Library

一个功能强大的 ComfyUI 资产库管理插件，帮助您轻松管理和组织生成的图片、视频、音频和工作流。

## ✨ 功能特点

### 资产浏览
- 📸 **图片** - 支持 PNG、JPG、JPEG、WebP、BMP、GIF
- 🎬 **视频** - 支持 MP4、MOV、AVI、WebM、MKV
- 🎵 **音频** - 支持 MP3、WAV、FLAC、OGG、M4A
- 📜 **工作流** - 支持 JSON 工作流文件

### 预览功能
- 点击资产打开全屏预览模态框
- 图片支持缩放查看
- 视频支持播放控制
- 音频支持播放器

### 工作流集成
- 📂 **打开工作流** - 从 PNG 图片元数据中提取并加载工作流
- 📷 **加载图片** - 直接创建 LoadImage 节点并加载选中图片
- 🖼️ **自定义封面** - 为工作流设置自定义封面图片

### 文件夹管理
- 📁 创建/删除工作流文件夹
- 🔄 拖放工作流到文件夹进行分类
- 🏷️ 侧边栏文件夹标签快速筛选

## 📦 安装

1. 将此文件夹复制到 `ComfyUI/custom_nodes/` 目录
2. 重启 ComfyUI
3. 点击顶部菜单栏的资产库图标即可使用

## 📁 项目结构

```
ComfyUI-Asset-Library/
├── __init__.py          # 插件入口

│   ├── main.js          # 前端主逻辑
│   ├── style.css        # 样式表

├── py/
│   ├── __init__.py
│   └── api.py           # 后端 API

```

## 🔧 开发

### 环境要求
- ComfyUI (最新版本)
- Python 3.10+
- Node.js (可选，用于前端开发)

### 本地开发
1. Fork 并克隆仓库
2. 在 ComfyUI 的 custom_nodes 目录下创建符号链接
3. 修改代码后刷新浏览器即可看到效果
